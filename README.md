## <img src="https://api.iconify.design/lucide/activity.svg?color=%2310B981" width="24" height="24" style="vertical-align: middle;"> Monitoramento 24/7

Utilizamos **Grafana + Prometheus** para metricas detalhadas e **Uptime Kuma** para monitoramento de disponibilidade.

<img width="1879" height="625" alt="image" src="https://github.com/user-attachments/assets/0ef9f1e2-411d-4bf6-a7cd-3a84f70addaf" />

- <img src="https://api.iconify.design/lucide/layout-dashboard.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Grafana:** Dashboards com metricas de CPU, memoria, rede e disco em tempo real
- <img src="https://api.iconify.design/lucide/database.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Prometheus:** Coleta e armazenamento de metricas de todos os servicos, com retencao de 30 dias
- <img src="https://api.iconify.design/lucide/box.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **cAdvisor:** Metricas detalhadas de cada container Docker (CPU, memoria, I/O de rede e disco)
- <img src="https://api.iconify.design/lucide/cpu.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Node Exporter:** Metricas do servidor host - CPU por core, memoria total/disponivel, disco e rede
- <img src="https://api.iconify.design/lucide/heart-pulse.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Uptime Kuma:** Monitoramento de uptime e alertas automaticos. Acompanhe em **https://status.norhytale.com**

## <img src="https://api.iconify.design/lucide/hard-drive.svg?color=%2310B981" width="24" height="24" style="vertical-align: middle;"> Backups

Seus dados estao seguros conosco. Realizamos backups automaticos diarios com duas camadas de protecao:

- <img src="https://api.iconify.design/lucide/server.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Backup Local:** Armazenado no proprio servidor
- <img src="https://api.iconify.design/lucide/cloud.svg?color=%2310B981" width="16" height="16" style="vertical-align: middle;"> **Backup na Nuvem:** Enviado automaticamente para o Google Drive

Os backups sao mantidos por 7 dias, entao sempre temos como recuperar o progresso da comunidade caso algo de errado.
